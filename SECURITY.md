# Security Policy

## Supported Versions

Only the latest version of Fynancials is supported. The latest version will always be the HEAD of `main`.
If you find a vulnerability in a version predating the one on `main`, please check if the vulnerability still exists in the current version.

In addition, you are very welcome to report any vulnerabilitis you find in unmerged branches.

## Reporting a Vulnerability

Kindly hand in a [Security Advisory](https://github.com/aslopek/fynancials/security/advisories).
